*INTRODUCTION TO H.T.M.L {A CODING / PROGRAMMING LANGUAGE}*

* Full Form Of HTML *

The HTML stands for Hyper Text Markup Language . In which , Hyper Text means a type of text that is can't be understood by normal people and written to tell something to computer . And Markup Language means it is a Language that is easy for the computers to understand.

*Invention of HTML*

HTML is invented by Tim Berners Lee in the year of 1995 as a low standard language . But now , it is devloped as HTML 5 ,a much popular high standard language .

* Basics about HTML *

HTML is a well-known coding / programming language used to write codes and make various types of websites, games , apps ,etc...

* Features of HTML *

  • HTML is a non- case sensitive Language.
  • HTML codes are written with the help of tags.
  • Starting tags in HTML are enclosed in < > .
  • Endings tags are enclosed in < / > .

* Tags and its types*

As we all know tags are the most important thing in HTML because without them codes are can't be written.
Tags can be categorised as per two things :-

  • As per start and end :- According to this , tags can be of two types . These are:-
    1] Starting Tags - These are the tags used to start the use of any particular tag. These tags are enclosed with < > .
    2] Ending tags - These are the tags used to end the use of any particular tag . These tags are enclosed with < / > .
  • As per containibility :- According to this , tags can be of two types . These are :-
    1] Container Tags - These are the tags that requires both starting and ending tags particularly . It also contains some data in it.
    2] Empty Tags - These are the tags that requires only starting tag but no ending tag. It doesn't contain any data in it.

* How to end a particular container tag *

  • Sometimes, The tag that started first should be ended at last.e.g html,body,etc..
  • If the work of any tag is not needed much then it should be ended wherever it's work gets finished. e.g bold,italic,etc...
  • Some tags are ended separately like title is ended after writing the title , head is ended after ending the title. e.g head section etc...
  • Empty tags are not ended at all. e.g line break , horizontal line , etc..

* Parts of HTML Code *

There are mainly three parts of a HTML code . These are : -

  • Tags :- Tags tells us the web browser how to display the given data.
  • Attributes :- Attributes are the other characteristics of data like text color , background color , font ,etc..
  • Elements :- Elements is the mixture of any tag and its co-related attributes.

* The way to write a basic HTML code *

  • Step 1 :- Start the code with writing {DOCtype HTML! } as it tells the browser the version of HTML we are using .
    NOTE:- {DOCtype HTML! } states HTML 5 .
  • Step 2 :- Then start the code with {html} as it states it is a HTML document .
  • Step 3 :- Open the {head} tag as it shows the starting of Heading section.
  • Step 4 :- Open the {Title} tag and then write a suitable title for the website and then close the {title} tag .
  • Step 5 :- Close the {Head} tag and open the (body) tag as it shows the starting of the main body of the website.
  • Step 6 :- Provide the data in body in the way you want by using several tags in body like {paragraph}, {lists} , {bold, italic,underline} etc..
  • Step 7:- Close the {body} tag and then {html} tag too.
  • Step 8:- Save the document with extension (.html or .htm) as per your version .

* Some most important tags used in HTML code*

* Heading tag (H1-H6) *

  • It is a container tag .
  • It is used to write headings of any topic.
  • It has numbering from 1(smallest) to 6 (largest).
  • It can have various attributes.

* Paragraph tag (P) *

  • It is also a container tag .
  • It is used to write a paragraph on any topic.
  • It can too have various attributes.

* Ordered list (OL) *

  • It is also a container tag.
  • It is use to make Ordered lists.
  • It can be serial wise in number form , alphabet form , roman numerals etc...
  • It can too have various attributes.

* Unordered list (UL) *

  • It is also a container tag.
  • It is use to make unordered lists.
  • It can be in the form of Bulleted points , arrows , etc...
  • It can too have various attributes.

* Some other basic tags used in HTML code *

*Line break Tag (BR)*
  • It is an empty tag.
  • It is used to break a line and go to next line.
  • It can have no attributes.
* Horizontal line tag (HR)*
  • It is also an empty tag.
  • It is used to build a line horizontely on the screen .
  • It can also have attributes like length , width , shade / noshade , etc...
*Bold tag (B)*
  • It is a container tag .
  • It is used to bold any statement or part of statement.
  • It doesn't have any attributes.
* Underline tag (U)*
  • It is also a container tag .
  • It is used to underline any statement or part of statement.
  • It also doesn't have any attributes.
*Italic tag (I)*
  • It is also a container tag .
  • It is used to italicise any statement or part of statement.
  • It also doesn't have any attributes.

    The procees of linking any website to another website or some other part in the same websites is called as Hyperlinking .

      1] Internal linking - In this any part of website is been linked to other part of the same website .
